# Advanced Machine Learning — Course Notebooks

Companion notebooks for the **Advanced Machine Learning** course on Zylo.

Each notebook is named `{sectionId}-b{blockIndex}.ipynb` and contains
the runnable code from a single section of the course. They are
auto-generated from the course content and opened via the
"Run on Colab" button next to each code block in the Zylo learning app.

This course is written as **primary learning material**, not revision.
Sections are long and detailed — definitions, worked examples, pitfalls,
spaced quizzes, and deep dives into the theory.
